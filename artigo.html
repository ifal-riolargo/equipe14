DESENVOLVIMENTO WEB PARA DISPOSITIVOS MÓVEIS COM GRAILS


Fabiano Silva de Carvalho, Marissol Martins Barros, Fabio Rodrigues dos Santos, Helder Ribeiro Galvão

RESUMO

Esse artigo foi criado com o objetivo de disseminar conhecimentos sobre o desenvolvimento de aplicações para dispositivos móveis com tecnologias web utilizando Grails.
Grails é um framework focado na construção de aplicações web de forma mais ágil e produtiva através da utilização do paradigma de programação por convenção (que
preserva o desenvolvedor dos detalhes de configurações) e da linguagem de programação Groovy, que roda sobre a JVM. Ele combina os principais frameworks utilizados
na plataforma Java, como o Spring e Hibernate. O desenvolvimento de aplicações para dispositivos móveis vem crescendo muito em todo o mundo. Por causa desse
crescimento, e pela dificuldade de se manter compatibilidade com as diversas tecnologias dos dispositivos existentes, que surgiu uma nova abordagem de desenvolvimento
web para dispositivos móveis. Será realizada a seguir uma breve apresentação de como utilizar o Grails para desenvolver aplicativos móveis na web.
Palavras-chave:Groovy; Dinâmica; Grails; Framework; Web; Java; Tablet; Touch.

GROOVY ON GRAILS


Fabiano Silva de Carvalho, Marissol Martins Barros, Haline Anteria de Souza Moreira, Renato Machado de Araujo, Rodrigo de Oliveira Borges

RESUMO


Este artigo foi desenvolvido com o objetivo de elicitar e aglomerar informações a fim decompor e disseminar conhecimentos sobre a linguagem
de programação Groovy e o framework Grails. A linguagem de programação Groovy veio como auxílioao Java, trazendo inovações a fim de tornar
o desenvolvimento de softwares muito mais ágil e o produtivo. O Grails caracteriza-se em um framework web baseado em MVC (Padrão de Projeto) que
combina os principais frameworks utilizados na plataforma Java e respeita o "paradigma" Convention-over-configuration (programação por convenção). A
seguir será realizada uma breve apresentação do assunto de forma a esclarecer conceitos e teorias com o objetivo de enfocar na inovação provocada por essas tecnologias.
Palavras-chave: Dinâmica; Framework; Integração; Groovy; Grails; Java. Ver artigo completo
