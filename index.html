Sobre o curso:

Grails é um framework para construção de aplicações para web através da linguagem de programação Groovy (uma linguagem dinâmica para a plataforma Java).
Nossa empressa tem como objetivo aumentar a produtividade ensinando a criar framework web para a plataforma java, usando tecnologias maduras do mundo java.
